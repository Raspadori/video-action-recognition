# Human Action Recognition in Videos using Deep Learning

## Description
This project performs human action recognition from video sequences using Deep Learning. Video frames are processed with OpenCV and classified using a PyTorch model to recognize actions such as walking, running, and jumping through preprocessing, training, inference, and visualization.

## Technologies
- Python
- PyTorch
- OpenCV
- NumPy
- Matplotlib
- Google Colab

## Dataset
UCF101

## Setup

Install:

pip install -r requirements.txt

Run notebook.

## Workflow
Video → Frame Extraction → Training → Classification → Visualization
